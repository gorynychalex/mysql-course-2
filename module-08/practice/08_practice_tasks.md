# Практические задания - Модуль 8

## Задание 1. Резервное копирование

**Цель:** Научиться создавать резервные копии

### Задание:

1. Создайте полную резервную копию базы данных
2. Создайте копию только структуры
3. Создайте копию только данных
4. Восстановите базу из резервной копии

---

## Задание 2. Управление пользователями

**Цель:** Освоить создание и управление пользователями

### Задание:

1. Создайте пользователя только для чтения
2. Создайте пользователя для операций с данными
3. Создайте администратора базы
4. Проверьте права каждого пользователя

---

## Задание 3. Оптимизация запросов

**Цель:** Научиться оптимизировать запросы

### Задание:

1. Используйте EXPLAIN для анализа 5 запросов
2. Найдите запросы с полным сканированием таблицы
3. Добавьте необходимые индексы
4. Сравните EXPLAIN до и после оптимизации

---

## Задание 4. Экспорт данных

**Цель:** Научиться экспортировать данные

### Задание:

1. Экспортируйте таблицу в CSV
2. Экспортируйте таблицу в XML
3. Экспортируйте таблицу в HTML
4. Создайте JSON-отчёт

---
