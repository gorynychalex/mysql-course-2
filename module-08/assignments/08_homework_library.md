# Домашнее задание - Модуль 8

## Тема: Обслуживание и оптимизация библиотечной системы

### Легенда

Библиотечная система запущена в production. Необходимо обеспечить её надёжность, безопасность и производительность.

---

## Часть 1. Резервное копирование (10 баллов)

### Задание:

1. **Создайте скрипт резервного копирования:**
   - Полная копия базы данных
   - Копия только структуры
   - Инкрементальная копия данных

2. **Настройте автоматическое копирование:**
   - Ежедневное полное копирование
   - Ежечасное копирование транзакций

3. **Протестируйте восстановление:**
   - Восстановление из полной копии
   - Восстановление из инкрементальной

---

## Часть 2. Безопасность и пользователи (10 баллов)

### Задание:

Создайте пользователей с правами:

| Пользователь | Права | Описание |
|-------------|-------|----------|
| `librarian_read` | SELECT | Чтение для сотрудников |
| `librarian_write` | SELECT, INSERT, UPDATE | Работа с выдачами |
| `librarian_admin` | ALL кроме DROP | Администрирование |
| `backup_user` | SELECT, LOCK TABLES, RELOAD | Для резервного копирования |
| `report_user` | SELECT на представления | Для отчётов |

---

## Часть 3. Оптимизация (15 баллов)

### Задание:

1. **Анализ запросов:**
   - Выполните EXPLAIN для 10 запросов
   - Найдите проблемные запросы
   - Задокументируйте проблемы

2. **Оптимизация:**
   - Добавьте необходимые индексы
   - Перепишите неоптимальные запросы
   - Создайте представления для частых запросов

3. **Сравнение:**
   - Покажите EXPLAIN до и после
   - Замерьте время выполнения

---

## Часть 4. Мониторинг (5 баллов)

### Задание:

1. Включите slow_query_log
2. Настройте логирование запросов дольше 1 секунды
3. Создайте отчёт по медленным запросам
4. Настройте мониторинг размера таблиц
