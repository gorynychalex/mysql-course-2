# Домашнее задание - Модуль 6

## Тема: Автоматизация библиотечных операций

### Легенда

Необходимо автоматизировать рутинные операции библиотеки с помощью хранимых процедур, функций и триггеров.

---

## Часть 1. Хранимые процедуры (15 баллов)

### Задание:

Создайте процедуры:

1. **`sp_issue_book`** - Выдача книги читателю
   - Проверка доступности копии
   - Проверка отсутствия блокировок у читателя
   - Создание записи о выдаче
   - Обновление статуса копии

2. **`sp_return_book`** - Возврат книги
   - Обновление записи о выдаче
   - Расчёт и создание штрафа (если просрочено)
   - Обновление статуса копии

3. **`sp_reserve_book`** - Резервирование книги
   - Проверка возможности резерва
   - Создание резерва
   - Уведомление (запись в лог)

4. **`sp_get_reader_history`** - История читателя
   - Все выдачи читателя
   - Сумма штрафов
   - Статистика

---

## Часть 2. Хранимые функции (10 баллов)

### Задание:

Создайте функции:

1. **`fn_calculate_age`** - Возраст читателя по дате рождения
2. **`fn_calculate_fine`** - Расчёт штрафа по датам
3. **`fn_is_book_available`** - Доступность книги (BOOLEAN)
4. **`fn_get_reader_status`** - Статус читателя (VARCHAR)
5. **`fn_format_currency`** - Форматирование суммы в рубли

---

## Часть 3. Триггеры (15 баллов)

### Задание:

Создайте триггеры:

1. **`before_loan_insert`** - Проверка перед выдачей
   - Читатель не должен быть заблокирован
   - Не более 5 активных выдач

2. **`after_loan_insert`** - После выдачи
   - Логирование в audit_log
   - Обновление статистики читателя

3. **`before_reader_insert`** - Перед добавлением читателя
   - Автоматическая установка даты регистрации
   - Генерация номера билета

4. **`after_fine_insert`** - После создания штрафа
   - Отправка уведомления (запись в таблицу notifications)

---

## Часть 4. События (EVENTS) (5 баллов)

### Задание:

Создайте события:

1. **Ежедневная проверка просрочек** - обновление статуса выдач
2. **Еженедельная очистка** - архивация старых записей
3. **Ежемесячный отчёт** - агрегация статистики
