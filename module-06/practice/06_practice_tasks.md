# Практические задания - Модуль 6

## Задание 1. Хранимые процедуры

**Цель:** Научиться создавать и использовать процедуры

### Задание:

1. Создайте процедуру для выдачи книги читателю
2. Создайте процедуру для возврата книги
3. Создайте процедуру для получения статистики книги
4. Создайте процедуру для начисления штрафа

---

## Задание 2. Хранимые функции

**Цель:** Научиться создавать функции

### Задание:

1. Функция расчёта возраста читателя
2. Функция определения статуса книги (доступна/выдана)
3. Функция расчёта суммы задолженности читателя
4. Функция форматирования ФИО

---

## Задание 3. Триггеры

**Цель:** Освоить создание триггеров

### Задание:

1. Триггер для автоматического обновления даты возврата
2. Триггер для логирования изменений в выдачах
3. Триггер для проверки возраста читателя (минимум 14 лет)
4. Триггер для обновления статистики после выдачи

---

## Задание 4. Встроенные функции

**Цель:** Изучить встроенные функции MySQL

### Задание:

Используйте в запросах:
- 5 математических функций
- 5 строковых функций
- 5 функций даты/времени
- 5 функций работы с NULL
