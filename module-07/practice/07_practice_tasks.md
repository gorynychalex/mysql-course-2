# Практические задания - Модуль 7

## Задание 1. Транзакции

**Цель:** Научиться использовать транзакции

### Задание:

1. Создайте транзакцию для выдачи книги
2. Создайте транзакцию для возврата книги со штрафом
3. Продемонстрируйте откат транзакции
4. Используйте SAVEPOINT для частичного отката

---

## Задание 2. Блокировки

**Цель:** Освоить блокировки данных

### Задание:

1. Используйте LOCK TABLES для критической секции
2. Используйте SELECT ... FOR UPDATE
3. Используйте GET_LOCK для именованной блокировки
4. Продемонстрируйте разницу между READ и WRITE блокировками

---

## Задание 3. Уровни изоляции

**Цель:** Понять уровни изоляции транзакций

### Задание:

1. Проверьте текущий уровень изоляции
2. Продемонстрируйте проблему при READ UNCOMMITTED
3. Покажите решение с SERIALIZABLE
4. Объясните выбор уровня для библиотечной системы

---

## Задание 4. Хранилища

**Цель:** Изучить типы хранилищ

### Задание:

1. Создайте таблицы с разными движками (InnoDB, MyISAM, MEMORY)
2. Сравните производительность
3. Объясните выбор движка для каждой таблицы библиотеки

---
