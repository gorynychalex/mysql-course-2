# Домашнее задание - Модуль 7

## Тема: Транзакционная безопасность библиотеки

### Легенда

Система библиотеки должна гарантировать целостность данных при одновременной работе нескольких сотрудников.

---

## Часть 1. Транзакции для операций (15 баллов)

### Задание:

Создайте хранимые процедуры с транзакциями:

1. **`sp_safe_issue_book`** - Безопасная выдача книги
   - Блокировка записи читателя
   - Проверка лимита выдач
   - Создание выдачи
   - Обновление статуса копии
   - Логирование операции

2. **`sp_safe_return_book`** - Безопасный возврат
   - Расчёт штрафа
   - Создание штрафа (если нужно)
   - Обновление выдачи
   - Обновление копии

3. **`sp_transfer_book`** - Перемещение между филиалами
   - Обновление локации копии
   - Логирование перемещения
   - Обновление статистики филиалов

---

## Часть 2. Блокировки (10 баллов)

### Задание:

Реализуйте блокировки:

1. Используйте `SELECT ... FOR UPDATE` для предотвращения гонок
2. Используйте `GET_LOCK` для блокировки книги на время оформления
3. Создайте процедуру с `LOCK TABLES` для массовой операции
4. Документируйте случаи использования блокировок

---

## Часть 3. Обработка ошибок (10 баллов)

### Задание:

1. Используйте `DECLARE HANDLER` для обработки ошибок
2. Реализуйте откат при ошибках
3. Логируйте ошибки в таблицу `error_log`
4. Возвращайте понятные сообщения об ошибках
