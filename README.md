# MySQL: Проектирование и создание баз данных

**Объем:** 32 академических часа

## Описание курса

Курс предназначен для изучения проектирования, создания и администрирования баз данных на основе СУБД MySQL. Слушатели освоят теоретические основы реляционных баз данных, научатся создавать схемы, писать SQL-запросы, разрабатывать хранимые процедуры и триггеры, а также оптимизировать работу сервера.

## Структура курса

| Модуль | Тема | Часы |
|--------|------|------|
| 1 | Установка и запуск сервера MySQL | 4 |

| 3 | Синтаксис выражений SQL для определения структуры данных | 4 |
| 4 | Выражения SQL: манипулирование данными | 4 |
| 5 | Объединение запросов и манипулирование данными | 4 |

| 8 | Оптимизация и обслуживание сервера MySQL | 4 |

## Требования к слушателям

- Базовые навыки работы с командной строкой
- Основы работы с файловой системой
- Понимание принципов работы с данными

## Результаты обучения

После завершения курса слушатели смогут:

- Устанавливать и настраивать сервер MySQL
- Проектировать реляционные базы данных
- Создавать и модифицировать структуру БД
- Выполнять CRUD-операции с данными
- Писать сложные SQL-запросы с объединениями и подзапросами
- Разрабатывать хранимые процедуры и триггеры
- Работать с транзакциями
- Оптимизировать запросы и обслуживать сервер БД

## Система оценки

- **Практические задания:** 40%

## Финальный проект

Разработка полноценной системы управления библиотекой с использованием всех изученных технологий MySQL.
