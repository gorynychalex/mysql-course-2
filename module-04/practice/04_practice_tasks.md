# Практические задания - Модуль 4

## Задание 1. CRUD операции

**Цель:** Отработать базовые операции CRUD

### Задание:

1. **CREATE (INSERT):**
   - Добавьте 5 новых читателей в таблицу `readers`
   - Добавьте 3 новые книги в таблицу `books`
   - Добавьте 2 новых автора в таблицу `authors`

2. **READ (SELECT):**
   - Выведите всех читателей с фамилией на букву "И"
   - Выведите книги, изданные после 2010 года
   - Выведите авторов, у которых более 1 книги

3. **UPDATE:**
   - Обновите email у одного из читателей
   - Увеличьте цену всех книг на 10%
   - Измените статус книги на 'archived' для книг старше 50 лет

4. **DELETE:**
   - Удалите читателя, который не имеет активных выдач
   - Удалите книги со статусом 'archived'

### Отчет:
- SQL-скрипт с выполнением операций
- Скриншоты результатов

## Задание 2. Сложные SELECT запросы

**Цель:** Научиться писать сложные запросы выборки

### Задание:

Напишите запросы для получения следующей информации:

1. Список всех книг с именами авторов (JOIN)
2. Читатели, у которых есть активные выдачи (EXISTS)
3. Книги, которые никогда не выдавались (NOT EXISTS)
4. Топ-5 самых выдаваемых книг
5. Средняя длительность выдачи по категориям книг
6. Количество книг каждого автора
7. Читатели, у которых нет штрафов
8. Книги с ценой выше средней по библиотеке

### Отчет:
- SQL-скрипт с 8 запросами
- Результаты выполнения

---

## Задание 3. Предикаты и фильтрация

**Цель:** Освоить различные предикаты SQL

### Задание:

Создайте запросы с использованием:

1. **LIKE:** Поиск книг по шаблону названия
2. **BETWEEN:** Книги в ценовом диапазоне
3. **IN:** Книги определённых категорий
4. **IS NULL:** Читатели без телефона
5. **REGEXP:** Поиск по регулярному выражению
6. **EXISTS:** Проверка существования связанных записей

### Отчет:
- SQL-скрипт с примерами
- Пояснения к каждому предикату

---

## Задание 4. JOIN операции

**Цель:** Научиться объединять таблицы

### Задание:

Создайте запросы с разными типами JOIN:

1. **INNER JOIN:** Книги с авторами
2. **LEFT JOIN:** Все читатели с их выдачами (включая тех, у кого нет выдач)
3. **RIGHT JOIN:** Все выдачи с информацией о книгах
4. **SELF JOIN:** Иерархия категорий (родительские и дочерние)
5. **Multiple JOIN:** Полная информация о выдаче (читатель + книга + автор)

### Отчет:
- SQL-скрипт с запросами
- Объяснение разницы между типами JOIN

## Задание 5. Агрегатные функции и группировка

**Цель:** Освоить GROUP BY и агрегатные функции

### Задание:

Напишите запросы:

1. Количество книг по категориям
2. Средняя цена книг по годам издания
3. Общий количество выдач по каждому читателю
4. Сумма штрафов по статусам
5. Максимальная, минимальная и средняя длительность выдачи
6. Количество авторов с книгами в каждой категории
7. Месячная статистика выдач (группировка по месяцам)
8. Топ-10 читателей по количеству прочитанных книг

### Отчет:
- SQL-скрипт с запросами
- Результаты выполнения
