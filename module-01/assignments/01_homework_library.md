## Часть 2. Создание базы данных (10 баллов)

### Задание:

Создайте базу данных `library_db` со следующей структурой:

#### Таблица `books` (книги):
| Поле | Тип | Описание |
|------|-----|----------|
| id | INT, PRIMARY KEY, AUTO_INCREMENT | ID книги |
| title | VARCHAR(255), NOT NULL | Название книги |
| author | VARCHAR(100) | Автор |
| isbn | VARCHAR(20) | ISBN |
| year_published | INT | Год издания |
| pages_count | INT | Количество страниц |
| created_at | TIMESTAMP | Дата добавления |

#### Таблица `readers` (читатели):
| Поле | Тип | Описание |
|------|-----|----------|
| id | INT, PRIMARY KEY, AUTO_INCREMENT | ID читателя |
| first_name | VARCHAR(50), NOT NULL | Имя |
| last_name | VARCHAR(50), NOT NULL | Фамилия |
| email | VARCHAR(100) | Email |
| phone | VARCHAR(20) | Телефон |
| registration_date | DATE | Дата регистрации |

#### Таблица `categories` (категории книг):
| Поле | Тип | Описание |
|------|-----|----------|
| id | INT, PRIMARY KEY, AUTO_INCREMENT | ID категории |
| name | VARCHAR(100), NOT NULL | Название категории |
| description | TEXT | Описание |

### Требования:
- Все таблицы должны использовать движок InnoDB
- Кодировка таблиц: utf8mb4_unicode_ci
