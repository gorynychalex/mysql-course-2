# Практические задания - Модуль 5

## Задание 1. UNION операции

**Цель:** Научиться объединять результаты запросов

### Задание:

1. Объедините списки читателей и сотрудников библиотеки
2. Найдите книги, которые есть в наличии или зарезервированы
3. Создайте общий список всех выдач и резервирований с датами
4. Объедините поиск по названиям книг и именам авторов

### Отчет:
- SQL-скрипт с UNION запросами
- Объяснение разницы UNION vs UNION ALL

---

## Задание 2. Подзапросы

**Цель:** Освоить подзапросы в WHERE, FROM, SELECT

### Задание:

Напишите запросы с подзапросами:

1. **IN:** Книги авторов из определённой страны
2. **NOT IN:** Читатели без задолженностей
3. **Сравнение:** Книги дороже средней цены
4. **В FROM:** Статистика по категориям
5. **В SELECT:** Количество выдач для каждой книги

---

## Задание 3. EXISTS

**Цель:** Научиться использовать EXISTS для проверок

### Задание:

1. Найдите категории, у которых есть книги
2. Найдите книги, у которых есть свободные экземпляры
3. Найдите читателей, у которых есть активные выдачи
4. Найдите книги, которые никогда не выдавались
5. Найдите авторов без книг в библиотеке

---

## Задание 4. Представления

**Цель:** Научиться создавать и использовать VIEW

### Задание:

Создайте представления:

1. `v_available_books` - доступные книги
2. `v_overdue_loans` - просроченные выдачи
3. `v_reader_stats` - статистика читателей
4. `v_popular_books` - популярные книги (топ-20)
5. `v_new_arrivals` - новые поступления (за месяц)

Проверьте работу каждого представления.
